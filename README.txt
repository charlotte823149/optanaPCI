使用說明:
1. 先點選[檔案]->[讀取資料夾]選擇需要的資料夾
2. 點選[圖名]選擇需要顯示的照片檔
3. 選擇破壞類型及等級並在圖片上畫出直線或多邊形
4. 按下ctrl+S儲存圖片
5. 重複1.~4.直到資料夾內檔案都修改完成
6. 按下ctrl+S儲存最後一張照片資訊
7. 按下[匯出]製作excel報表

圖片大小與基底切換:
輸入"長"與"寬"後按下Enter鍵即可設定圖片大小並且重新計算基底
有五種基底可供選擇
!!! 更改基底後必須在重新看過所有照片後才會重新計算PCI !!!

快捷鍵(必須切換為英文輸入法才可使用):
ctrl+S: 儲存當前圖片的破壞資訊
ctrl+C: 按下後即可使用方向鍵"左"、"右"來切換上一張或下一張圖片
ctrl+Z: 返回上一步(畫圖的部分)
Enter: 按下Enter鍵即可在同樣破壞、同樣等級的情況下開始繪製新的破壞(其餘時候只需切換破壞種類及等級就會自動新增新的破壞)
"+": 放大圖片
"-": 縮小圖片
insert: 選取[特殊道路現況]中的類別及數量再按下insert鍵即可加入到最右欄的表格中
delete: 選取最右欄表格中欲刪除的項目並按下delete鍵即可刪除
F1~F12: 切換破壞種類
